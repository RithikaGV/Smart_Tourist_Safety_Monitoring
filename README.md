# Smart Tourist Safety Monitoring & Incident Response System using AI, Geofencing, and Blockchain

The **Smart Tourist Safety Monitoring & Incident Response System** is an intelligent safety platform designed to enhance the security of tourists, especially in remote and low-network areas. By integrating **Artificial Intelligence (AI), Blockchain, GPS Geofencing, Edge Computing, and Bluetooth Mesh networking**, the system provides real-time safety monitoring, secure digital identity management, and rapid emergency response while preserving user privacy.

Tourists complete a secure KYC-based onboarding process, after which a tamper-proof blockchain-based digital ID is generated for the duration of their trip. The application continuously monitors the user's location using AI-powered geofencing and detects entry into restricted or hazardous areas, even without an internet connection. It also identifies unusual route deviations and sends safety prompts when necessary.

In emergencies, the app features a one-touch SOS mechanism that can relay distress messages through nearby devices using Bluetooth Mesh and Wi-Fi Direct when cellular networks are unavailable. The system prioritizes user privacy by encrypting location history and allowing access only during authorized emergencies or legal investigations.

For authorities, a centralized dashboard provides live risk heatmaps, dynamic danger zone management, AI-based movement pattern analysis, blockchain-backed incident records, and automated E-FIR generation to streamline emergency response and improve public safety.

This project demonstrates the practical application of modern technologies to build a reliable, secure, and privacy-focused tourist safety ecosystem capable of operating effectively in both online and offline environments.
